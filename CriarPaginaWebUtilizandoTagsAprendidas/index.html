O que é HTML?

HTML é uma linguagem de marcação de hipertexto que possui um conjunto de tags responsáveis pela marcação do conteúdo de uma página no navegador.

Como utilizar HTML?

Através da tags.

Quando utilizar HTML?

Utilizamos o HTML para estruturar uma página web e seu conteúdo.

O que são tags?

Tags são os elementos a mais que escrevemos usando a sintaxe < e > e o nomedatag. Diversas tags são disponibilizadas pela linguagem HTML e cada uma possui uma funcionalidade específica.

Como utilizar tags?

Uma tag é definida com caracteres , e seu nome. Muitas tags possuem conteúdo, como o texto do título. Nesse caso, para determinar onde o conteúdo acaba, usamos uma tag de fechamento com a barra antes do nome: nomedatag. Algumas tags podem receber algum tipo de informação extra dentro de sua definição chamada de atributo. São parâmetros usando a sintaxe de atributo="valor".

Quando utilizar tags?

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

O que são atributos?

Atributos são as particularidades, qualidades e características que são próprias de alguém ou algo. Existem dois tipos pricipais de atributos: atributos de classe que pertencem a classe em si e atributos de instância que pertecem a cada objeto individualmente.

Como utilizar atributos?

Geralmente os atributos possuem tipo como int, char, boolean, etc. Chave = valor

Quando utilizar atributos?

Quando quiser apresentar uma característica ou particularidade de alguém ou algo.

O que são textos?

Um texto é uma manifestação da linguagem. Pode ser definido como tudo aquilo que é dito por um emissor e interpretado por um receptor.

Como utilizar textos?

Em HTML utilizamos varias tags para representar e escrever textos, como por exemplo: h1 a h6 para representar títulos de vários tamanhos, a tag p que representa um paragráfo, blockquote para citações ou observações. Há combinações para diferenciar pelavras, como a tag strong que representa o negrito, sublinhado com a tag u de underline e i para itálico. Marca texto utilizamos a tag mark. E a tag sup serve para colocarmos algo que seja elevado, ou resgistro de marca e asterísco e a tah sub para colocar abaixo da escrita.

Quando utilizar textos?

Quando houver a necessidade de apresentar uma manisfestação de linguagem através da escrita.

O que são listas ordenadas e não ordenadas?

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementose e a lista não-ordenada coloca marcadores antes de cada elemento.

Como utilizar listas ordenadas e não ordenadas?

Através das tags, as tags usadas para criar uma lista ordenada são ol e li. A tag ol é usada para definir a lista e a tag li é usada para cada item da lista.

Quando utilizar listas ordenadas e não ordenadas?

Quando quiser listar itens utilizando ou não marcadores.

O que são links?

Link significa um canal de direcionamento para um site na internet, ou seja, uma palavra, texto ou imagem que quando é clicada pelo usuário, o encaminha para outra página na internet, que pode conter outros textos ou imagens.

Como utilizar links?

Através da tag de âncora, a.

Quando utilizar links?

Quando precisamos indicar que um trecho de texto se refere a um outro conteúdo, seja ele no mesmo documento ou em outro endereço (por exemplo uma página na web).
